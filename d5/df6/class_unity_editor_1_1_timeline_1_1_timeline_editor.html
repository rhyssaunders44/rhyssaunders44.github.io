static void UnityEditor.Timeline.TimelineEditor.Refresh ( RefreshReason  reason)
static

Refreshes the different components affected by the currently inspected UnityEngine.Timeline.TimelineAsset, based on the RefreshReason provided.

For better performance, it is recommended that you invoke this method once, after you modify the UnityEngine.Timeline.TimelineAsset. You should also combine reasons using the | operator.

Note: This operation is not synchronous. It is performed during the next GUI loop.
